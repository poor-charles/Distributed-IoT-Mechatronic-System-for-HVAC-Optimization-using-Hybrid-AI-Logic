# Digital Twin (EnergyPlus) Framework for HVAC Energy Performance Assessment

## Overview

This repository contains the complete implementation of an EnergyPlus-based Digital Twin developed for HVAC energy performance assessment through virtual parallel operation. The framework integrates Building Energy Modeling (BEM), Bayesian calibration, independent validation, MQTT-based real-time telemetry, and operational dashboard visualization.

The Digital Twin was developed to reproduce the operational behavior of the target conference room and enable continuous comparison between measured and simulated building performance. The framework supports both energy-performance assessment and comfort-related evaluation through real-time monitoring of indoor temperature, relative humidity, and Predicted Mean Vote (PMV).

## Digital Twin Development Workflow

### Script 1 – Data Preprocessing

Prepares and cleans the measured datasets collected from the physical building. Sensor data are synchronized, formatted, and converted into analysis-ready datasets.

### Script 2 – EnergyPlus Schedule Generation

Generates occupancy, thermostat setpoint, and workshop temperature schedules required for EnergyPlus simulation.

### Script 3 – Baseline Model Assessment

Evaluates the performance of the uncalibrated Building Energy Model by comparing simulated results against measured building data.

### Script 4 – Bayesian Optimization Parameter Space Definition

Defines the calibration parameters and allowable search ranges used by the Bayesian optimization framework.

### Script 5 – EnergyPlus Parameter Injection and Simulation Run

Automatically modifies EnergyPlus model parameters based on candidate solutions generated by the optimizer and executes simulation runs.

### Script 6 – Calibration Scoring and Model Performance Evaluation

Calculates calibration metrics including RMSE, CV(RMSE), and NMBE for HVAC energy consumption, indoor temperature, and relative humidity.

### Script 7 – Bayesian Calibration Optimization Loop

Implements Gaussian Process Regression (GPR)-based Bayesian Optimization to identify the parameter set that best matches measured building performance.

### Script 8 – Calibration Results Extraction and Visualization

Extracts the champion calibrated model and generates calibration comparison plots, scatter plots, and residual analyses.

### Script 9 – Independent Validation Data Preparation

Processes unseen operational data and prepares validation schedules and measured datasets.

### Script 10 – Independent Validation Simulation

Executes the calibrated Digital Twin using independent validation data.

### Script 11 – Validation Results Analysis and Visualization

Calculates validation metrics and generates validation time-series plots, scatter plots, and residual histograms.

### Script 12 – Operational Digital Twin Dashboard

Provides a Streamlit-based dashboard for real-time visualization of building performance and Digital Twin outputs.

### Script 13 – Real-Time Operational Digital Twin Engine

Implements live synchronization between the physical asset and EnergyPlus through MQTT communication, enabling real-time virtual parallel assessment.

### Operational Digital Twin Launcher

Automatically launches the Operational Digital Twin Dashboard and Real-Time Digital Twin Engine to support deployment and real-time virtual parallel assessment.

## Calibration and Validation Metrics

Calibration Performance:

* HVAC Energy CV(RMSE): 29.96%
* HVAC Energy NMBE: 9.96%

Validation Performance:

* HVAC Energy CV(RMSE): 28.23%
* HVAC Energy NMBE: 7.64%

Both calibration and validation satisfied the ASHRAE Guideline 14 acceptance criteria for hourly energy calibration.
