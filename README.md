# Distributed-IoT-Mechatronic-System-for-HVAC-Optimization-using-Hybrid-AI-Logic
A Bayesian Optimized Random Forest model integrated with deterministic physics based safety guardrails to control HVAC operation modes, hunting for the operational settings that minimizes energy consumption while maintaining comfort based on ASHRAE 55 Standards.

-> The Repo contains all the necessary files, programs, and data along with instructions to replicate the system. 
-> Necessary adjustments in the programs such as file locations/directories indicated in the program globals MUST BE CHANGED accordingly.
